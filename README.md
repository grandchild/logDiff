#### Usage
Paste log into left text field and click outside of it to (re)create diff on right side.

Time format can either be
- `YYYY-MM-DD HH:mm:ss` or
- `YYYY-MM-DD HH:mm:ss.SSSS` (any number of sub-second digits) or
- Unix timestamp
